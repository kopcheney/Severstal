# Severstal
Задание номер 4 на Python. <br>
Реализованы функции добавить нового пациента, записать на прием, получить список пациентов по номеру телефона, история приемов питомца, удаление питомца и истории его приемов, обновление статуса приема, изменение номера телефона владельца питомца. а так же вспомогательные функции. <br>
Библиотека Sqlite3 для бд, а так же PrettyTable для визуально приятного вывода, так как интерфейс реализован в консоли. <br>
